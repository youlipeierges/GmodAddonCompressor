# Garry's Mod 资源压缩工具
 - 原作者为: Shark vil
 - 翻译作者为 : YUKI
## 当前功能:

- 图像压缩 (支持 VTF, JPG, JPEG, PNG 格式)
- 音频文件压缩 (支持 WAV, MP3|DEMO, OGG|DEMO)
- LUA 代码最小化
